# SmartWarehouse
Smart Werehouse for RTC


## Installation

for prepare venv
```bash
  pip install -r 'requirements.txt'
```

For start localhost (backend)

```bash
  uvicorn SmartWarehouse.app.main:app --reload
```

For start localhost (frontend)

```bash
  cd ./SmartWarehouse/frontend
  npm run dev
```
