# SmartWarehouse
Smart Werehouse for RTC


## Installation

To prepare venv
```bash
  pip install -r 'requirements.txt'
```

To start localhost (backend)

```bash
  uvicorn app.main:app --reload
```

To start localhost (frontend)

```bash
  cd ./SmartWarehouse/frontend
  npm run dev
```
