# О проекте
Изначально этот сайт создавался для хакатона "Расти в IT", однако не был выполнен в срок. Этот проект представляет собой систему управления складом с достаточно широким набором возможностей.
# Как запустить проект?
Для запуска откройте проект в IDE и наберите команду:<br/>
`docker compose -f 'docker-compose.yml' up -d --build`<br/>
Предполагается, что на вашем компьютере установлен Docker. Если это не так, [то скачать его можно здесь](https://www.docker.com/)
# Какие функции есть на сайте?
1. Авторизация, необходимая для работы с остальными страницами. При этом доступных пользователей всего 3, их адреса электронной почты и пароли можно посмотреть в файле *app\models\users_methods.py*. Есть возможность запоминания пользователя.
2. Интерактивная карта склада, которую можно масштабировать. На ней схематично изображены роботы, можно посмотреть основную информацию о них.
3. Карточки со статистикой, обновляемые каждые 5 секунд
4. График активности роботов за последний час, обновляемый каждые 30 минут
5. Таблица последних сканирований с возможностью автообновления.
6. Просмотр истории инвентаризации. Можно фильтровать записи и экспортировать их в Excel и PDF.
7. Возможность прикрепить CSV-файл и загрузить информацию из него в базу данных. При этом файл автоматически проверяется на корректность. <br/>
